Seaside Getaway

CSS media queries for mobile-responsive elements and text/image-resizing across all visual breakpoints.


Some Insight Into the Design:


  • Color scheme - The Seaside Getaway site uses a complementary color scheme for a more appealing user experience. The soft earth tones coincide with site images and communicate a selling point. The complementary color scheme was picked using the hex color wheel.

  • Navigation - The website also uses strategic navigation to enhances user experience. By condensing elements into easy-to-find categories, viewers have no problem accessing the information that they need.

  • Load Time - Seaside Getaway has been optimized for browser load time using Google Chrome's developer tools. Several optimization methods include image resizing and file organization, all tested in the developer tool's network waterfall.
